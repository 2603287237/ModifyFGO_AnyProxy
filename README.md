## 使用教程
1. 在release下载最新版
2. 解压到自己喜欢的目录
3. 新建debug,trunfile,profile三个目录
4. 下载[AnyProxy](https://github.com/alibaba/anyproxy)
5. 解压到科技目录下
正确操作的目录树如下
-Folder
|-anyproxy
||-bin
||-...
|-debug
|-trunfile
|-profile
|-fgo.js
|-node.msi
|-README.md
|-run.bat
|-xfgo_v1.5.apk
6. 双击run.bat开启科技（可能提示没有安装node.js或nodemon，按照提示安装即可）
